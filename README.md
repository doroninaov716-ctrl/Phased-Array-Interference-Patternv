# Phased Array Interference Pattern

Этот репозиторий содержит скрипт, который вычисляет интерференционную картину от фазированной антенной решётки (phased array) и сохраняет её как изображение.

## Что делает код

- Располагает N эмиттеров концентрическими кольцами (каждое кольцо - N_ANGLE-угольник),
- Для каждого эмиттера задаётся амплитуда, длина волны и расстояние до экрана,
- Для каждой точки на экране численно интегрируется мгновенная интенсивность I(t, x, y) = (Σ_i A_i(t, x, y))^2 по времени на интервале T_INTEGR,
- Полученная временная интегральная яркость нормируется и отображается цветом (красный - яркость ≤ максимум, зелёный - отмеченные превышения),
- Также строится изображение расположения эмиттеров.

## Формулы и пояснения

- Волновое число: k = 2π / λ
- Угловая частота: ω = 2π f, где f = c / λ
- Путь от i-го излучателя до точки (x, y) на экране:
  L_i(x, y) = sqrt(l_i^2 + (x + r_i cos α_i)^2 + (y + r_i sin α_i)^2)
  где l_i — расстояние от излучателя до экрана, r_i и α_i — полярные координаты излучателя.
- Мгновенная амплитуда от i-го излучателя:
  A_i(t, x, y) = A_max_i * sin(ω_i t + k_i * L_i(x, y) + φ_i)
  где φ_i — первоначальная фаза (в коде φ выбирается так, чтобы i=0 был опорой).
- Мгновенная интенсивность:
  I(t, x, y) = (Σ_i A_i(t, x, y))^2
- В коде вычисляется временный интеграл:
  I_avg(x, y) = ∫_0^{T} I(t, x, y) dt
  (этот интеграл даёт меру средней яркости на интервале T)
